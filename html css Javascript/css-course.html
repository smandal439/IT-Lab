Module 2: Styling with CSS

Bringing the Web to Life

Cascading Style Sheets (CSS) is used to style and layout web pages. It can alter the font, color, size, and spacing of your content, split it into multiple columns, or add animations and other decorative features.

The Box Model

Everything in CSS is a box. The Box Model consists of margins, borders, padding, and the actual content.

.box {
  /* Content size */
  width: 200px;
  height: 100px;

  /* Space INSIDE the border */
  padding: 20px;

  /* The border itself */
  border: 2px solid blue;

  /* Space OUTSIDE the border */
  margin: 15px;
}

Flexbox Layout

Flexbox makes it easy to align elements perfectly horizontally or vertically.

.container {
  display: flex;
  justify-content: space-between; /* Spacing */
  align-items: center; /* Vertical alignment */
}

.item {
  flex: 1; /* Take up available space */
}
