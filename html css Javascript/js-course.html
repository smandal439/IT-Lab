Module 3: JavaScript Programming

Adding Logic to the Web

JavaScript is a powerful programming language that can calculate, manipulate, and validate data. It runs in the browser, making your pages highly interactive.

DOM Manipulation

The DOM (Document Object Model) is how JavaScript sees your HTML. You can use JavaScript to change HTML elements on the fly.

// Change text content
document.getElementById("title").textContent = "New Title!";

// Change styles
document.querySelector(".box").style.backgroundColor = "red";

// Create a new element
const newPara = document.createElement("p");
newPara.textContent = "I was created with JS.";
document.body.appendChild(newPara);

ES6+ Modern Syntax

Modern JavaScript has clean syntax features like Arrow Functions and Template Literals.

// Arrow Function
const add = (a, b) => a + b;

// Template Literals (using backticks)
const name = "Sarah";
console.log(`Welcome back, ${name}!`);
